# Animation System

Animations are spatial explanations, not decorative effects.

## Animation Tiers

none
primitive
smooth
elegant

## Philosophy

Animations should:
- reinforce continuity
- explain transitions
- avoid distraction
- degrade gracefully

## Performance Targets

Preferred FPS:
- 60 ideal
- 30 minimum

Animation durations:
- overlays: 120ms
- page transitions: 180–240ms
