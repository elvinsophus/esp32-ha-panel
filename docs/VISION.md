# Overall Vision

This project aspires to provide an operating system for ESP32-powered smart panels with an elegant, calm, spatially coherent user experience.

It is a product, not a toy, experiment, or temporary dashboard.

The panel is not merely a screen displaying Home Assistant entities. It should behave more like a spatial appliance interface or embedded operating system.

Primary goals:
- calmness
- elegance
- clarity
- responsiveness
- continuity
- spatial memorability
- modularity
- graceful degradation on weaker hardware
