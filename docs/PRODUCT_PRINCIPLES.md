# Product Principles

1. The root layer must remain calm.
2. Spatial memory is more important than information density.
3. Frequently used actions must require minimal interaction cost.
4. Layers should not feel cluttered.
5. Animations should reinforce spatial continuity.
6. The system must remain usable even if HA is unavailable.
7. Device profiles may alter presentation but not information architecture.
8. UI must degrade gracefully on weaker hardware.
9. Every visible element must justify its existence.
10. The panel should feel premium despite limited hardware resources.
