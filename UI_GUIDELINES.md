# UI Guidelines

## Design Philosophy

The interface should feel:

- calm
- premium
- modern
- elegant
- restrained

Avoid:
- excessive visual noise
- bright saturated colors
- clutter
- tiny unreadable widgets

---

# Preferred Characteristics

- generous spacing
- smooth transitions
- subtle animations
- dark-first theme
- readable typography
- clean geometry

---

# Inspirations

Desired inspiration direction:
- automotive HMIs
- premium smart-home panels
- modern industrial control systems
- minimalist operating systems

Avoid:
- gaming UI aesthetics
- cyberpunk overload
- RGB-heavy designs
